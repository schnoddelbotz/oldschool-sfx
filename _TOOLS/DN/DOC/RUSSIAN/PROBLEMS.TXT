INSTALL
-------
Для запуска инсталляции на Русском языке запустите:
 INSTALL.EXE -rus

Register
--------
 Для регистрации необходимо:
 - получить регистрационный ключ(процедура получения описана в readme.rus);
 - переименовать его в dn.key(если вы получили его под другим именем);
 - скопировать в стартовую директорию;
 - перезапустить DN - теперь Dos Navigator зарегистрирован, о чем он с
   радостью сообщит. Регистрация прописывается 'вовнутрь', так что
   регистрационный файл можно удалить(сохранив где-нибудь для дальнейших
   версий);
 - еще раз перезапустить уже зарегистрированный Dos Navigator.

Hackers registrations
---------------------
Если Dos Navigator играет похоронный марш, не запускает внешние программы
и сам не выгружается из памяти, то это значит, что Вы пытаетесь использовать
Dos Navigator с НЕЛЕГАЛЬНЫМ ключом! Удалите хакерский DN.KEY из вашей
директории и обратитесь к ближайшему дилеру для регистрации Вашей копии
Dos Navigator'а. Аналогичное сообщение Dos Navigator должен выдавать при
запуске. При этом НИКАКИХ ДЕСТРУКТИВНЫХ ОПЕРАЦИЙ НЕ ПРОИЗВОДИТСЯ, просто
Dos Navigator вежливо предупреждает Вас о том, что кто-то пытался его
нелегально модифицировать.

Install language file in User directory
---------------------------------------
 Необходимо установить Modify AUTOEXEC.BAT, так как необходимо
 прописать переменную окружения DNDLG.
