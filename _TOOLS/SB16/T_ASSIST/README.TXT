1.  Texto'LE

    Während der Installation von TextAssist können Sie eine Verknüpfung
    aller Textdateien (*.TXT) mit Texto'LE einrichten. Falls Sie dies getan 
    haben, jedoch zu einem späteren Zeitpunkt die Verknüpfung von Texto'LE 
    mit *.TXT-Dateien aufheben möchten, gehen Sie folgendermaßen vor:

       (a) Bearbeiten Sie die Datei TEXTOLE.INI in Ihrem 
           Windows-Verzeichnis
       (b) Fügen Sie unter dem Abschnitt [Settings] folgende Zeile hinzu:

            [Settings]
            TXT=0

       (c) Starten Sie den Datei-Manager, und verknüpfen Sie *.TXT-Dateien 
           mit Notepad oder einem beliebigen anderen Texteditor.
	    
    Sobald Sie Windows das nächste Mal starten, versucht Texto'LE nicht 
    mehr automatisch, sich selbst mit *.TXT-Dateien zu verknüpfen.

    Sie können Ihre Textdateien erneut mit Texto'LE verknüpfen,
    indem Sie die Zeile "TXT=0" aus der Datei TEXTOLE.INI entfernen.
